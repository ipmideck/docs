title: "Lenovo XCC"
description: "Включите IPMI-over-LAN на Lenovo XClarity Controller (XCC / XCC2), чтобы ipmideck мог подключиться к вашему серверу ThinkSystem."
---

**Применимо к:** серверы Lenovo ThinkSystem с XClarity Controller (XCC) и XCC2.

<Note>
  Lenovo XCC, **Экспериментальный** производитель для ipmideck, эти шаги взяты из официальной документации
  Lenovo и реального XCC (ThinkSystem SR630 V2). IPMI-over-LAN **отключён по
  умолчанию** на XCC в целях безопасности, и Lenovo рекомендует снова отключать его, когда он не
  используется.
</Note>

## Шаги

<Steps>
  <Step title="Откройте веб-интерфейс XCC">
    В браузере перейдите по IP-адресу XCC и войдите. Учётные данные по умолчанию:
    **USERID** / **PASSW0RD** (обратите внимание на **ноль** в "PASSW0RD"). Измените их немедленно,
    если они всё ещё используются.

    <Frame caption="The Lenovo XClarity Controller login screen (default USERID / PASSW0RD).">
      <img src="/tutorials/LENOVO/1_20221028032221331.jpg" alt="Lenovo XClarity Controller login screen on a ThinkSystem SR630 V2" />
    </Frame>
  </Step>
  <Step title="Откройте BMC Configuration → Network">
    Из панели XCC откройте **BMC Configuration → Network** в левой навигации.

    <Frame caption="From the XCC dashboard, open BMC Configuration → Network.">
      <img src="/tutorials/LENOVO/2_20221028033229267.jpg" alt="Lenovo XCC dashboard with BMC Configuration and Network highlighted" />
    </Frame>
  </Step>
  <Step title="Включите IPMI по LAN">
    Под **Service Enablement and Port Assignment** установите **IPMI over LAN** на **Enabled**
    (используется порт **623**).

    <Frame caption="Service Enablement and Port Assignment: toggle IPMI over LAN on (port 623).">
      <img src="/tutorials/LENOVO/3_20221028033746252.jpg" alt="Lenovo XCC Service Enablement showing IPMI over LAN enabled on port 623" />
    </Frame>
  </Step>
  <Step title="Проверьте связь">
    С хоста, который будет запускать ipmideck:

    ```bash
    ipmitool -I lanplus -H <xcc-ip> -U USERID -P <password> chassis status
    ```
  </Step>
</Steps>

<Note>
  **Пометка для проверки:** XCC2 (более новое поколение) использует тот же путь, но с переработанным
  веб-интерфейсом. Проверьте навигацию на вашем реальном оборудовании.
</Note>
