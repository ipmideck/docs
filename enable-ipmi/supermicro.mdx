<Warning>
  **Low-confidence guide — verify on your hardware before relying on these steps.**
  Supermicro BMC firmware differs significantly between board generations (X8 through X12).
  The screenshots below are from an older Supermicro BMC and BIOS; treat the navigation
  paths as a starting point, confirm them against your own board, and report corrections.
  Owner hardware validation is pending.
</Warning>

**Applies to:** Supermicro servers with an IPMI 2.0 BMC (X8 / X9 / X10 / X11 / X12 boards).

<Note>
  Supermicro is **Experimental** with ipmideck — IPMI 2.0 works but is not yet exhaustively
  tested. On most Supermicro boards **IPMI-over-LAN is enabled by default**; the main task
  is giving the BMC a reachable LAN address, either from the web interface or from BIOS.
</Note>

## Option A — from the BMC web interface

<Steps>
  <Step title="Log in to the BMC web interface">
    In a browser, navigate to the BMC IP address. The default credentials are
    **ADMIN** / **ADMIN** — change them immediately after first access.

    <Frame caption="The Supermicro BMC login screen (default ADMIN / ADMIN).">
      <img src="/tutorials/supermicro/IPMI-login.png" alt="Supermicro BMC web interface login screen" />
    </Frame>
  </Step>
  <Step title="Open Configuration → Network">
    Open the **Configuration** tab and choose **Network** from the options list. (This older
    firmware groups everything under *Configuration*; newer boards may label it
    *Network → Network Settings*.)

    <Frame caption="The Configuration menu — open Network to set the BMC's LAN address.">
      <img src="/tutorials/supermicro/IPMI-web-configuration.png" alt="Supermicro Configuration options menu" />
    </Frame>
  </Step>
  <Step title="Set the BMC's network address">
    Choose **Obtain an IP address automatically (DHCP)** or **Use the following IP address**
    and enter a static IP, subnet mask, and gateway, then click **Save**. IPMI-over-LAN is
    already active on the default channel — this just makes the BMC reachable.

    <Frame caption="Network Settings — DHCP or a static IP, subnet mask, and gateway.">
      <img src="/tutorials/supermicro/IPMI-network-configuration.png" alt="Supermicro Network Settings page with static IP configuration" />
    </Frame>
  </Step>
</Steps>

## Option B — from BIOS

If the BMC has no address yet, set it from BIOS. Press **DEL** during POST to enter the
BIOS Setup Utility.

<Steps>
  <Step title="Advanced → IPMI Configuration">
    On the **Advanced** tab, open **IPMI Configuration**.

    <Frame caption="BIOS Advanced tab — select IPMI Configuration.">
      <img src="/tutorials/supermicro/BIOS-advanced.png" alt="Supermicro BIOS Advanced settings menu" />
    </Frame>
  </Step>
  <Step title="Set LAN Configuration">
    Confirm **Status Of BMC: Working**, then open **Set LAN Configuration**.

    <Frame caption="IPMI Configuration — BMC status and Set LAN Configuration.">
      <img src="/tutorials/supermicro/IPMI-configuration.png" alt="Supermicro BIOS IPMI Configuration screen" />
    </Frame>
  </Step>
  <Step title="Open the LAN channel">
    Under **LAN Configuration** (Channel 01) you can set the **IP Address**, **Subnet Mask**,
    and **Gateway Address**.

    <Frame caption="LAN Configuration — IP Address, Subnet Mask, and Gateway Address sub-menus.">
      <img src="/tutorials/supermicro/LAN-configuration.png" alt="Supermicro BIOS LAN Configuration menu" />
    </Frame>
  </Step>
  <Step title="Set a static IP, subnet, and gateway">
    Set **IP Address Source** to **Static**, then enter the IP, subnet mask, and gateway.
    Press **F10** to save and exit.

    <Frame caption="IP Address Configuration — set the source to Static and enter the address.">
      <img src="/tutorials/supermicro/IP-configuration.png" alt="Supermicro BIOS IP Address configuration" />
    </Frame>
    <Frame caption="Subnet Mask configuration.">
      <img src="/tutorials/supermicro/Subnetmask-configuration.png" alt="Supermicro BIOS Subnet Mask configuration" />
    </Frame>
    <Frame caption="Default Gateway configuration.">
      <img src="/tutorials/supermicro/Gateway-configuration.png" alt="Supermicro BIOS Default Gateway configuration" />
    </Frame>
  </Step>
</Steps>

## Verify

From the host that will run ipmideck:

```bash
ipmitool -I lanplus -H <bmc-ip> -U ADMIN -P ADMIN chassis status
```

<Note>
  **Flag for review:** Supermicro BMC firmware varies a lot across X8–X12 generations and
  these screenshots are from an older board. Newer boards (X11 / X12) have a redesigned web
  UI. Test against your specific board before relying on this guide.
</Note>
