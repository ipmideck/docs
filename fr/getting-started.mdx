## Qu'est-ce qu'ipmideck ?

ipmideck est une plateforme de gestion IPMI auto-hébergée et basée sur le web. Elle se connecte aux
BMC (Baseboard Management Controller) de vos serveurs via IPMI 2.0 et vous offre une surveillance
des capteurs en temps réel, un contrôle intelligent des courbes de ventilateurs (FanPilot), une gestion
de l'alimentation à distance et des journaux d'événements matériels, le tout depuis votre navigateur, sans CLI.

ipmideck s'exécute entièrement sur votre propre réseau. Il n'y a pas de cloud, pas de télémétrie et
aucune dépendance externe : il dialogue directement avec vos BMC et stocke tout en local.

<Note>
  ipmideck doit atteindre vos BMC sur le réseau via le port UDP 623 (le port IPMI/RMCP+).
  Lorsque vous l'exécutez dans Docker, utilisez `--network host` pour que le conteneur puisse atteindre
  directement les BMC de votre LAN.
</Note>

## Démarrage rapide

Le moyen le plus rapide d'essayer ipmideck est Docker :

```bash
docker run -d --name ipmideck --network host \
  -v ipmideck-data:/data \
  devluigi06/ipmideck:latest
```

Le volume `-v ipmideck-data:/data` conserve votre configuration, votre base de données et votre
clé d'identifiants entre les redémarrages et les mises à niveau du conteneur. Sous Windows ou
macOS, remplacez `--network host` par `-p 3000:3000`, consultez [Installation](/fr/installation).

Ouvrez ensuite `http://<your-ip>:3000` dans un navigateur et suivez l'assistant de configuration pour ajouter votre
premier serveur.

Vous préférez les paquets Python ? Installez plutôt depuis PyPI :

```bash
pip install ipmideck
```

Le paquet pip nécessite l'installation d'`ipmitool` sur l'hôte.

<Note>
  Sur Debian 12+ et Ubuntu 23.04+, `pip install` échoue avec
  `error: externally-managed-environment` (PEP 668). Utilisez plutôt pipx, consultez
  [Installer sur Debian et Ubuntu](/fr/installation#installer-sur-debian-et-ubuntu-pep-668).
</Note>

## Ce que vous obtenez

- **Surveillance des capteurs** : relevés en direct de température, RPM des ventilateurs, tension et puissance, avec jusqu'à
  un an d'historique.
- **FanPilot** : un éditeur de courbes de ventilateurs en glisser-déposer avec une boucle de contrôle autonome et sûre.
- **Contrôle de l'alimentation** : On, Soft Off, Hard Off, Reset et Power Cycle avec un journal d'audit complet.
- **System Event Log (SEL)** : parcourez et exportez le journal d'événements matériels de votre BMC.
- **Inventaire matériel (FRU)** : numéros de série, références et données du fabricant.
- **Tableau de bord multiserveur** : gérez de nombreux BMC depuis une seule instance.
